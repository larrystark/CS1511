Project Description

Programming Project 7

Consider the Bag ADT. You have already identified that it has two important data members The class has the following attributes:

  • n, an integer, (the number of items in the bag)
  • an array of strings (the contents of the bag)

and the following member functions

  • add() - to add an item to the bag
  • show() - to list all of the items in the bag

You may have also implemented a remove() function

  • remove() - to remove an item from the bag

To fulfill all requirements for this program you must implement the following public class methods:

  • remove() - to remove an item from the bag
  • isEmpty() - to see whether the bag is empty.
  • clear() - to empty the bag

The main change in this implementation of the Bag ADT is that you will also implement another class (the Item ADT). Instead of the data implementation being an array of strings, make it an array of Item objects.

This means that your public member functions of the Bag ADT will need to be revised to handle Item objects, not just strings.

Use the main program provided and the Coding Challenge from this week to complete the project.
